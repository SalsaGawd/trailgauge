# TrailGauge

Off-road inclinometer with trip recording, weather, and elevation tracking. Built for trail driving and overlanding.

## Features

- Artificial horizon and analog dial inclinometer modes
- Pitch and roll readout with slope grade %
- Tilt alarm with adjustable threshold
- GPS coordinates, elevation, heading
- Live weather + hourly forecast with wind direction
- Daylight countdown (sunset / sunrise timing)
- Trip recorder with distance, elevation gain, moving time
- GPX export for use with Gaia, onX, Google Earth
- Auto-recovery of in-progress trips after app close
- Works in portrait and landscape
- Dark and light themes

## Status

Web prototype. Native iOS and Android versions in development.
